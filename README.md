# Discord Productivity & Utility Bot

## Features

SQLite database to persist tasks, reminders, and user data.

Reminders: Set one-time reminders with flexible time formats (10m, 2h, 1d) and get DMed when due.

Task management: Add, delete, and list tasks; mark them complete.

Bad word filter: Moderation tools to detect and block offensive words.

Role management: Assign roles to users via commands.

Epic Games tracker: Displays free games of the week.

Stock data: Fetch stock information using the yfinance API.

Secret commands: Hidden or admin-only features for advanced control.

Planned features: Weather reports, more productivity tools, and additional integrations.
